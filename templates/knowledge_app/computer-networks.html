📖 什么是计算机网络

计算机网络是将分布在不同地理位置的计算机通过通信线路连接起来，实现资源共享和信息传递的系统。它是现代信息社会的基础设施，支撑着互联网、移动通信、云计算等各种应用。

学习计算机网络不仅要理解各种协议的工作原理，还要掌握网络设计、性能优化、安全防护等实践技能。

🏗️ OSI七层模型

OSI（开放系统互连）模型将网络通信分为七个层次，每层负责特定的功能，为网络协议设计提供了标准框架。

📡 重要协议族

🌐 TCP/IP协议族
  • 📍IP (Internet Protocol)：网络层协议，负责数据包的路由和寻址
  • 🔗TCP (Transmission Control Protocol)：可靠的面向连接传输协议
  • ⚡UDP (User Datagram Protocol)：快速的无连接传输协议
  • 🔍ICMP (Internet Control Message Protocol)：网络诊断和错误报告
🌍 应用层协议
  • 🌐HTTP/HTTPS：Web浏览和安全传输协议
  • 📧SMTP/POP3/IMAP：电子邮件传输和接收协议
  • 📁FTP/SFTP：文件传输协议
  • 🏷️DNS：域名解析系统
🛣️ 路由协议
  • 🗺️RIP (Routing Information Protocol)：距离向量路由协议
  • 🔗OSPF (Open Shortest Path First)：链路状态路由协议
  • 🌐BGP (Border Gateway Protocol)：边界网关协议
