⟷ 双端队列 (Deque - Double-ended Queue)

两端都能操作的强大数据结构，集栈和队列功能于一体

📋

知识点概述

核心概念：双端队列是一种允许在前端（front）和后端（back）都可以进行插入和删除操作的线性数据结构，是栈和队列的泛化。

⬅️ 前端操作

addFirst() - 前端插入
removeFirst() - 前端删除

➡️ 后端操作

addLast() - 后端插入
removeLast() - 后端删除

🔍 查看操作

peekFirst() - 查看前端
peekLast() - 查看后端

⚡ 高效性

所有基本操作都是O(1)时间复杂度

高级前置知识：栈、队列、数组或链表基础操作

🔍

概念详解

🚪 生活类比：双开门电梯

想象一个两边都有门的电梯：

  • 前门 = front端（可以进人，也可以出人）
  • 后门 = back端（也可以进人，也可以出人）
  • 乘客 = 数据元素
  • 灵活进出 = 可以从任一端进出，适应不同场景

💡 关键：双端队列提供了最大的操作灵活性！

🆚 与栈和队列的关系

📚 栈 (Stack)

LIFO

  • 只能一端操作
  • push/pop
  • 受限的双端队列

🚀 队列 (Queue)

FIFO

  • 两端各司其职
  • enqueue/dequeue
  • 受限的双端队列

⟷ 双端队列 (Deque)

灵活

  • 两端都可操作
  • 四种基本操作
  • 栈+队列的功能

🎯 四种核心操作

⬅️ 前端操作

  • addFirst(x)：在前端插入元素x
  • removeFirst()：删除并返回前端元素
  • peekFirst()：查看前端元素但不删除

➡️ 后端操作

  • addLast(x)：在后端插入元素x
  • removeLast()：删除并返回后端元素
  • peekLast()：查看后端元素但不删除
⚠️ 初学者易混淆点：
  • 理解"双端"：不是两个队列，而是一个队列的两端都能操作
  • 操作对称性：addFirst对应removeLast，addLast对应removeFirst
  • 实现复杂度：需要同时维护前后两个指针或索引
  • 空队列判断：当size为0或front与back相遇时

#include <iostream>
#include <vector>
using namespace std;

template<typename T>
class Deque {
private:
    vector<T> data;
    int frontIndex;
    int backIndex;
    int size;
    int capacity;

public:
    Deque(int cap = 10) {
        capacity = cap;
        data.resize(capacity);
        frontIndex = 0;
        backIndex = 0;
        size = 0;
    }

    // 前端插入
    bool addFirst(T element) {
        if (isFull()) {
            cout << "双端队列已满!" << endl;
            return false;
        }

        frontIndex = (frontIndex - 1 + capacity) % capacity;
        data[frontIndex] = element;
        size++;
        cout << "前端插入: " << element << endl;
        return true;
    }

    // 后端插入
    bool addLast(T element) {
        if (isFull()) {
            cout << "双端队列已满!" << endl;
            return false;
        }

        data[backIndex] = element;
        backIndex = (backIndex + 1) % capacity;
        size++;
        cout << "后端插入: " << element << endl;
        return true;
    }

    // 前端删除
    T removeFirst() {
        if (isEmpty()) {
            cout << "双端队列为空!" << endl;
            return T();
        }

        T element = data[frontIndex];
        frontIndex = (frontIndex + 1) % capacity;
        size--;
        cout << "前端删除: " << element << endl;
        return element;
    }

    // 后端删除
    T removeLast() {
        if (isEmpty()) {
            cout << "双端队列为空!" << endl;
            return T();
        }

        backIndex = (backIndex - 1 + capacity) % capacity;
        T element = data[backIndex];
        size--;
        cout << "后端删除: " << element << endl;
        return element;
    }

    // 查看前端元素
    T peekFirst() {
        if (isEmpty()) {
            cout << "双端队列为空!" << endl;
            return T();
        }
        return data[frontIndex];
    }

    // 查看后端元素
    T peekLast() {
        if (isEmpty()) {
            cout << "双端队列为空!" << endl;
            return T();
        }
        return data[(backIndex - 1 + capacity) % capacity];
    }

    // 判断是否为空
    bool isEmpty() {
        return size == 0;
    }

    // 判断是否已满
    bool isFull() {
        return size == capacity;
    }

    // 获取大小
    int getSize() {
        return size;
    }

    // 显示双端队列
    void display() {
        if (isEmpty()) {
            cout << "双端队列为空" << endl;
            return;
        }

        cout << "双端队列内容: ";
        for (int i = 0; i < size; i++) {
            int index = (frontIndex + i) % capacity;
            cout << data[index] << " ";
        }
        cout << endl;
    }
};
import java.util.ArrayList;

public class Deque<T> {
    private ArrayList<T> data;
    private int frontIndex;
    private int backIndex;
    private int size;
    private int capacity;

    public Deque(int capacity) {
        this.capacity = capacity;
        this.data = new ArrayList<>(capacity);
        for (int i = 0; i < capacity; i++) {
            this.data.add(null);
        }
        this.frontIndex = 0;
        this.backIndex = 0;
        this.size = 0;
    }

    // 前端插入
    public boolean addFirst(T element) {
        if (isFull()) {
            System.out.println("双端队列已满!");
            return false;
        }

        frontIndex = (frontIndex - 1 + capacity) % capacity;
        data.set(frontIndex, element);
        size++;
        System.out.println("前端插入: " + element);
        return true;
    }

    // 后端插入
    public boolean addLast(T element) {
        if (isFull()) {
            System.out.println("双端队列已满!");
            return false;
        }

        data.set(backIndex, element);
        backIndex = (backIndex + 1) % capacity;
        size++;
        System.out.println("后端插入: " + element);
        return true;
    }

    // 前端删除
    public T removeFirst() {
        if (isEmpty()) {
            System.out.println("双端队列为空!");
            return null;
        }

        T element = data.get(frontIndex);
        frontIndex = (frontIndex + 1) % capacity;
        size--;
        System.out.println("前端删除: " + element);
        return element;
    }

    // 后端删除
    public T removeLast() {
        if (isEmpty()) {
            System.out.println("双端队列为空!");
            return null;
        }

        backIndex = (backIndex - 1 + capacity) % capacity;
        T element = data.get(backIndex);
        size--;
        System.out.println("后端删除: " + element);
        return element;
    }

    // 查看前端元素
    public T peekFirst() {
        if (isEmpty()) {
            System.out.println("双端队列为空!");
            return null;
        }
        return data.get(frontIndex);
    }

    // 查看后端元素
    public T peekLast() {
        if (isEmpty()) {
            System.out.println("双端队列为空!");
            return null;
        }
        return data.get((backIndex - 1 + capacity) % capacity);
    }

    // 判断是否为空
    public boolean isEmpty() {
        return size == 0;
    }

    // 判断是否已满
    public boolean isFull() {
        return size == capacity;
    }

    // 获取大小
    public int getSize() {
        return size;
    }

    // 显示双端队列
    public void display() {
        if (isEmpty()) {
            System.out.println("双端队列为空");
            return;
        }

        System.out.print("双端队列内容: ");
        for (int i = 0; i < size; i++) {
            int index = (frontIndex + i) % capacity;
            System.out.print(data.get(index) + " ");
        }
        System.out.println();
    }
}
class Deque:
    """双端队列类"""

    def __init__(self, capacity=10):
        self.capacity = capacity
        self.data = [None] * capacity
        self.front_index = 0
        self.back_index = 0
        self.size = 0

    def add_first(self, element):
        """前端插入"""
        if self.is_full():
            print("双端队列已满!")
            return False

        self.front_index = (self.front_index - 1 + self.capacity) % self.capacity
        self.data[self.front_index] = element
        self.size += 1
        print(f"前端插入: {element}")
        return True

    def add_last(self, element):
        """后端插入"""
        if self.is_full():
            print("双端队列已满!")
            return False

        self.data[self.back_index] = element
        self.back_index = (self.back_index + 1) % self.capacity
        self.size += 1
        print(f"后端插入: {element}")
        return True

    def remove_first(self):
        """前端删除"""
        if self.is_empty():
            print("双端队列为空!")
            return None

        element = self.data[self.front_index]
        self.front_index = (self.front_index + 1) % self.capacity
        self.size -= 1
        print(f"前端删除: {element}")
        return element

    def remove_last(self):
        """后端删除"""
        if self.is_empty():
            print("双端队列为空!")
            return None

        self.back_index = (self.back_index - 1 + self.capacity) % self.capacity
        element = self.data[self.back_index]
        self.size -= 1
        print(f"后端删除: {element}")
        return element

    def peek_first(self):
        """查看前端元素"""
        if self.is_empty():
            print("双端队列为空!")
            return None
        return self.data[self.front_index]

    def peek_last(self):
        """查看后端元素"""
        if self.is_empty():
            print("双端队列为空!")
            return None
        return self.data[(self.back_index - 1 + self.capacity) % self.capacity]

    def is_empty(self):
        """判断是否为空"""
        return self.size == 0

    def is_full(self):
        """判断是否已满"""
        return self.size == self.capacity

    def get_size(self):
        """获取大小"""
        return self.size

    def display(self):
        """显示双端队列"""
        if self.is_empty():
            print("双端队列为空")
            return

        result = []
        for i in range(self.size):
            index = (self.front_index + i) % self.capacity
            result.append(str(self.data[index]))

        print("双端队列内容:", " ".join(result))

if __name__ == "__main__":
    deque = Deque(6)

    # 测试各种操作
    deque.add_last(10)
    deque.add_last(20)
    deque.add_first(5)
    deque.add_first(1)

    deque.display()

    print(f"前端元素: {deque.peek_first()}")
    print(f"后端元素: {deque.peek_last()}")

    deque.remove_first()
    deque.remove_last()

    deque.display()

    print(f"队列大小: {deque.get_size()}")
    print(f"是否为空: {deque.is_empty()}")
    print(f"是否已满: {deque.is_full()}")

🔍 深度案例：工作窃取调度

在多线程程序中，双端队列是工作窃取调度算法的核心：

  • 🏭 工作线程：从自己队列的一端取任务（通常是队尾）
  • 🔄 窃取线程：从其他线程队列的另一端窃取任务（通常是队头）
  • ⚡ 减少竞争：两端操作减少了线程间的锁竞争
  • 📊 负载均衡：空闲线程能够从忙碌线程那里分担工作
  • 🎯 性能优化：Java的ForkJoinPool就使用了这种机制

学习检验

🤔 问题1：双端队列如何实现栈的功能？

答案：只使用双端队列的一端进行操作
实现方式：
  • 方案1：只使用前端 - addFirst()作为push，removeFirst()作为pop
  • 方案2：只使用后端 - addLast()作为push，removeLast()作为pop
  • 效果：实现了栈的LIFO（后进先出）特性
  • 优势：一个数据结构同时具备栈和队列功能

🤔 问题2：双端队列如何实现队列的功能？

答案：一端插入，另一端删除
实现方式：
  • 标准方案：addLast()作为enqueue，removeFirst()作为dequeue
  • 反向方案：addFirst()作为enqueue，removeLast()作为dequeue
  • 效果：实现了队列的FIFO（先进先出）特性
  • 选择建议：通常选择标准方案，符合直觉

🤔 问题3：对空的双端队列执行：addFirst(1), addLast(2), addFirst(3), removeLast(), removeFirst()，最终结果？

答案：队列中只剩下元素1
步骤分析：
  • 初始：[] (空队列)
  • addFirst(1)：[1]
  • addLast(2)：[1, 2]
  • addFirst(3)：[3, 1, 2]
  • removeLast()：[3, 1] (删除了2)
  • removeFirst()：[1] (删除了3)

🤔 问题4：双端队列相比单独使用栈和队列的优势是什么？

主要优势：
  • 统一接口：一个数据结构提供栈和队列的所有功能
  • 内存效率：不需要维护两个不同的数据结构
  • 操作灵活：可以根据需要在任一端操作，适应性强
  • 算法支持：某些算法（如滑动窗口最值）需要双端操作
  • 性能优化：避免数据在不同结构间复制转移

🤔 问题5：什么是滑动窗口最值问题，双端队列如何解决？

问题描述：给定数组和窗口大小k，求每个窗口的最大值
双端队列解法：
  • 存储策略：队列中存储数组索引，按值的大小排序
  • 维护单调性：保持队列中的值单调递减
  • 前端操作：removeFirst()移除过期的索引
  • 后端操作：removeLast()移除较小的值，addLast()添加新索引
  • 查询最值：peekFirst()得到当前窗口最大值的索引
  • 时间复杂度：O(n)，每个元素最多进出队列一次
